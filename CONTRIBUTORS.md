## Contributions

This project is maintained as a personal portfolio build. New contributors
are welcome — see [`CONTRIBUTING.md`](./CONTRIBUTING.md) for the workflow,
and [`docs/ARCHITECTURE.md`](./docs/ARCHITECTURE.md) for a tour of how
the pieces fit together.

### Recognised contribution areas

- **Code** — features, bug fixes, refactors, performance work, tests
- **Docs** — JSDoc, architecture notes, deployment guides, screenshots
- **Design** — UI/UX feedback, mockups, accessibility audits
- **Testing** — Playwright E2E, Lighthouse audits, manual QA reports
- **Triage** — issue clean-up, reproducibility checks

If you'd like to be listed here, send a PR adding yourself in the
relevant section. Use this format:

```markdown
- **Your Name** ([@github-handle](https://github.com/handle)) — short
  description of your contribution.
```
